Grouping Tags/ Tags de agrupamento

Temos duas tags de agrupamento, a div (box-level) e o span (inline-level).

Antigamente era muito usado div com id tipo cabeçalho, menu, etc. Isso fazia o documento ter várias divs dentro da outra.

Porém no html5 existem agrupamentos de tag (grouping tags) semânticas. Tráz vantagens na organização do meu conteúdo e também ajuda os mecânismos de busca.

Todo site tem um cabeçalho (header), um conteúdo principal (main) e um rodapé (footer).


    <body>
        <header>
            <h1>Meu site</h1>
            <nav>
                <a href="#">Link</a>
                <a href="#">Link</a>
                <a href="#">Link</a>
                <a href="#">Link</a>
                <a href="#">Link</a>
            </nav>
        </header>

        <main>

            <saction id="assunto">
                <article>
                    <aside>

                    </aside>
                </article>
            </saction>

            <saction id="noticias>

                <article>

                    <h2>Nóticia sobre ..</h2>
                    <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Laudantium exercitationem sit quidem, necessitatibus sed inventore, tenetur 
                    facere dolorem cupiditate reprehenderit iste, quo aspernatur architecto nihil rem optio quis! Totam, cum?</p>

                    <aside>
                        <p>Artigo escrito por.. (podemos colocar o nome do autor)</p>
                    </aside>

                </article>
                
            </saction>

        </main>

        <footer>
            Desenvolvido pelo CursoemVideo.
        </footer>
    </body>

Pra o menu dentro de um cabeçalho utilizamos o nav (navegação).

Dentro do main podemos ter uma seção section. Temos também o article (artigo) que é o artigo dentro de uma seção. O aside é um contéudo periférico/relativo ao artigo.

Depois é só formatar cada grouping tag com css. Veja o site formatado aqui usando esses agrupamento de tags.
