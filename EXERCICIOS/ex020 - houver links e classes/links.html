Pseudo-classes usados

No exemplo acima o a estilazação usa pseudo-classes. Uma pseudo-classe é usado para dizer que aquele elemento vai ter um estado específico e que pode ser estilizado quando tiver em certo estado:


       <style>

    body {
        font-family: Arial, Helvetica, sans-serif;
    }

    a {
        color: red;
        text-decoration: none;
        font-weight: bold;
    }

    a:visited {
        color: darkred;
    }

    a:hover {
        color: black;
        text-decoration: underline;
    }

    a:active {
        color: yellow;
    }
       </style>

Pseudo-elemento

Um pseudo-elemento permite que você estilize uma parte especifica de um elemento selecionado, você pode usar os pseudo-elementos junto as classes:


    <style>

    .especial {
        background-color: whitesmoke;
    }

    .especial::before {
        content: '❀ ';
    }
    .especial::after {
        content:' ❀';
    }

    </style>
