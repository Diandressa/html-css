Capítulo 1 - Estilos CSS

Capítulo 1.1 - Style Local/Interno

Primeiro colocar a tag style dentro de head. Agora usaremos seletores {}. Criar um seletor pra cada tag respectiva. A linha todo é chamada de declaração, que é o conjunto de propriedade e valor. Ex de declaração: color: red;

Capítulo 2 - Style em um doc css

Capítulo 2.1 - Style em um documento separado/ externo

Para criar um link, passe o mouse no valor da tag href e segura ctrl + clica.

Vai no estilo criado e da ctrl + x (só os seletores) para recortar e apaga a tag style dentro de head. Depois apaga o estilo nas outras páginas.

Em head digita link:css <link rel="stylesheet" href="style.css">

Cria esse link css nas outras páginas tb.

Segura o ctrl e clica sobre style.css para criar uma página de estilo e cola o que recortamos lá.

As vezes pode dar problemas de compatibilidades com acentuações então criamos uma regra na primeira linha. A regra é @charset "UTF-8";

Você pode ter mais de um arquivo css, 2 links css em head, que irá somar as modificações. Você também pode criar modificações locais (usando a tag style em head), links css e modificações inline juntos.
