Criando Sites com HTML

Usando o id com CSS

Para mudar uma tag em especifico no css, você pode dar uma identificação* pra ele:

        
        <h1 id="principal">

No CSS externo coloca:

        
        h1#principal {
            
        }

Em HTML é id = em CSS é # (cerquilha)/ Só pode um elemento
Em HTML é class = em CSS é . (ponto)/ Pode vários elementos

Dentro de um arquivo HTML só se pode ter um elemento com id. Por exemplo, se vc já usou id="principal" em um h1, você não pode usar o id="principal" em outro h1. (funciona, mas não é recomendado)

*<span>: O elemento HTML <span> é um conteiner generico em linha para conteúdo fraseado , que não representa nada por natureza. Ele pode ser usado para agrupar elementos para fins de estilo (usando os atributos class ou id ).

Usando Class

Quando você quer aplicar a mesma formatação em dois elementos aos mesmo tempo basta usar class. Você pode ter vários elementos com a mesma classe:

        
        <h1 class="explicacao1">Teste 1</h1>
        <h1 class="explicacao2">Teste 2</h1>
        <h1 class="explicacao3">Teste 3</h1>

        <h1 class="explicacao1">Teste 4</h1>
        <h1 class="explicacao2">Teste 5</h1>
        <h1 class="explicacao3">Teste 6</h1>

Nomeie o class conforme sua função e não sua forma. Ex: Título em verde: Curso básico, então seria lógico nomear o class como básico e não como verde.

Não coloque acentos nos class.

Aplicação no CSS (não precisa colocar o nomes, h1, h2...):

        
        .explicacao1 {

        }

        .explicacao2 {

        }

        .explicacao3 {

        }

Usando dois class na mesma tag.

Podemos usar dois class na mesma tag dessa forma: 

        
            <h1 class="explicacao1 destaque">Teste 1</h1>

No CSS:

        
            .explicacao1 {
            color: red;
            }

            .destaque {
            backgound-color: white;
            }

Aprendendo CSS

Podemos ter uma id e um class

Podemos usar um id e um class em uma mesma tag, porém o id sobrepoe.

        
            <h1 id="principal" class="explicacao1 destaque">Teste 1</h1>

O id tb sobrepoe as formatação do h1 no css
