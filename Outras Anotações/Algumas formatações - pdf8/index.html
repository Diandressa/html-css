Algumas formatações de textos

Semântica

No antigo HTML4 algumas tags representavam apenas formas, como <b>, <u> e <blink>. Não havia um motivo pra usá-las, era somente estético. Atualmente, no HTML5, a semântica (significado) é mais valorizada e não a forma (estética) que passou a ser configurado pelo CSS. Sendo assim, em HTML5, vemos os chamados HTML semântico ou tags semânticas ou ainda conteúdo semântico.

Você pode ver as tags obsoletes (em desuso) no site da w3.

Negrito e Itálico

A tag <strong> significa que o texto delimitado possui força dentro da frase e aparecerá em negrito.

Já a tag <em> significa que queremos dar ênfase (do Inglês emphasis) ao termo. Logo, aparecerá em itálico

Texto deletado

Antigamente utilizava-se a tag <strike> hoje não mais entre nós. Agora se usa o <del> que significa que o texto que está ali pode ser lido, mas deve ser desconsiderado pelo leitor.

Texto inserido

É o oposto do texto deletado. Se colocarmos um texto dentro de <ins> e </ins>, estamos dizendo que o texto que está ali, deve ser lido e você deve prestar atenção nele. (antigamente era usado a tag <u> para sublinhar um texto

Textos sobrescrito e subscrito

Para escrever x2 ou um H20 corretamente, basta usar as tags <sub> e <sup>. E então o texto ficará assim: x2 e H2O

Abreviações

Essa é uma novidade da HTML5. Sempre que você escrever uma sigla e quiser deixar claro ao usuário (e aos mecanismos de busca) o significado dela, use a tag <abbr> com parâmetro title para indicar o significado da sigla. Veja:

<p>Eu estou estudando <abbr title="Hyper Text Markup Langaue>HTML</abbr> para criar sites.</p>

Quando passamos o mouse sobre a sigla abreviada, um pequeno texto aparece com o seu significado.

Aplicação:
Eu estou estudando HTMLpara criar sites.

Texto Invertido

Para deixar um título ou um texto invertido use a tag <bdo>. BDO significa bi-directional override. Ao usar essa tag coloque também o parâmetro dir para iniciar uma das duas direções possíveis:

rtl = da direita para a esquerda (right-to-left)
ltr = da esquerda para a direita (left-to-right)

<h2>Texto Invertido</h2>
<bdo dir="rtl">Este texto todo está invertido no formato RIGHT-TO-LEFT.</bdo>

Aplicação:
Invertido rtl
Este texto todo está invertido no formato RIGHT-TO-LEFT

Invertido (jeito que lemos) ltr
Este texto todo está invertido no formato LEFT-TO-RIGHT
