Inserindo vídeos hospedados localmente

Este vídeo está hospedado no meu próprio servidor.

Colocar todos os formatos de vídeos, de preferência do mais leve pro mais pesado, para ser compátivel com todos os navegadores (não aparecerá todos os vídeos com as diferentes extenções, e sim um que o navegador suporte).

Colocar a tag vídeo e depois os source src dentro.

Para colocar uma imagem/tumb inseri o parâmetro poster

Porém dessa forma, do vídeo sendo hospedado no próprio navegador, há um grande consumo de banda e o custo é elevado para o cliente.

Inserindo vídeos do Youtube

Dessa forma não consome dados do seu servidor e sim do youtube, porém só se o vídeo for público.

Inserindo vídeo do Vimeo

Se quiser inserir um vídeo que só assinantes poderão ver no seu site (o youtube só permite quando o vídeo é público), então poderá usar o Vimeo
