Testando Gradiente

Para fazer gradiente no fundo usa-se a declaração (css) background-image.

Gradiente Linear

Usamos linear-gradient (a direção ou ângulo, a primeira cor, a próxima cor); Exemplo:

            
        body {
            backgound-image: linear-gradient (to right, white, blue);
            background-attachment: fixed;
        }

Obs.: Em to rigth a direção pode ser colocada em graus. Ex: 90deg seria 90 graus.

Para um fundo fixo inserir background-attachment: fixed;

Para direita usa-se to right, para esquerda to left, para cima to top* e para baixo to bottom.

* o gradiente em to top não fica como esperado, para isso precisamos fazer uma configuração global no CSS usando o *. Veja o exemplo:

Configuração Global

Usa-se o * (asterisco) 

            
        * { /* Confirações globais das CSS */
            height: 100%;
        }

Essa configuração global não deu certo no meu navegador

Gradiente Radial

Usamos o radial-gradient (circle, cor1, cor2);. Podemos colocar a porcentagem de determinada cor se preferirmos.

            
        body {
            backgound-image: radial-gradient (circle, white 10%, blue);
            background-attachment: fixed;
        }
