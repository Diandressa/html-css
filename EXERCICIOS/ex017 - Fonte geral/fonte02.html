Tipos de Alinhamento

Usamos o text-align

Configuramos o alinhamento no CSS, usando a declaração text-align:

        
        body {
            font: normal 16px Ariel, Verdana, serif;
            text-align: left; 
        }

        h1 {
            text-align: center;
        }

        h2 {
            text-align: right;
        }

        p {
            text-align: justify;
            text-indent: 30px;
        }

O text-indent é o espaçamento inicial no começo do parágrafo.
