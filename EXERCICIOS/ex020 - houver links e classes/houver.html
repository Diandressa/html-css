Parágrafo filho

Podemos dizer no css que um parágrafo (p) esta dentro de uma div usando o >

        
        div > p {
            display:none;
        }

Dessa forma, você consegue alterar só o parágrafo que está dentro da div.

Existe um parágrafo escondido nesse site.

Para fazer ele aparecer passando o mouse em cima do texto fazemos o seguinte:

        
        div:hover > p {
            display: block;
            color:white;
            background-color: violet;
            width: 200px;
        }

Podemos formatar o parágrafo dentro da div somente. O block é para mostrar o texto escondido.
