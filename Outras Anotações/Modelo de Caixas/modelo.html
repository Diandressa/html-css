Modelo de caixa/ Box model

Aninhamento: É um termo usado pra dizer que algo está dentro de outro. Gerando níveis de hierarquia.

Uma caixa tem um tamanho. Esse tamanho é especificado com:

height = altura
width = largura

A linha em volta dessa caixa/conteúdo é chamado de:

Border = borda

O espaçamento/acolchoamento dessa caixa é definida pelo:

Padding = Acolchoamento ou preenchimento

Ainda temos uma margem em volta da borda/border que é definida como:

Margin = margem

Obs.: o padding é o espaço interno do border/borda e o margin o espaço externo do border.

É possível criar um traço fora/envolvendo a borda pra isso é usado:

Outline = contorno ou traçado

O amarelo na imagem acima é o outline.

Tipos de caixas

Existem dois tipos de caixas:

box-level e inline-level

Box-level

Um box-level sempre se inicia em uma linha nova e sempre ocupa 100%, ou seja o espaço total, da minha tela/viewport.

Uma div é um exemplo de box-level.

Inline-level

Um inline-level não pula pra próxima linha, não quebra linha, continuando o conteúdo. É uma caixa dentro da linha.

Um span é um exemplo de inline-level. Ele não ocupa a linha toda, usamos quando queremos elementos próximos, elementos um do lado do outro.
