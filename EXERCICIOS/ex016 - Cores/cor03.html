A tag main

A tag main é como se fosse uma div, e diz que aquele contéudo é o principal.

Main no CSS

Abra link para um estilo separado para as css, nesse link abra uma declaração para main.

  • Faça um box de fundo branco com background-color: white; com largura width: 600px;
  • Preenchimento/acolchoamento padding: 10px;
  • Para centralizar margin: auto;

  • Para arredondar as pontas, em main configure uma declação desta forma, border-radius: 20px;

  • Para sombras use box-shadow: 5px 5px 15px white;. Há quatro parâmetros à serem definidos:
    1. O primeiro é quanto a sombra andará para o lado;
    2. O segundo é quanto a sombra andará para baixo;
    3. O terceiro é quanto ela irá espalhar;
    4. O último é a transparência/cor.

Configure os h1 e h2

Configure o h1 e h2 em um estilo css separado.

Para centralizar o texto use a declaração text-align: center;.

Para sombras em texto use text-shadow: 1px 1px 2px black
