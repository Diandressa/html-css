Código-fonte / Pré-formatação

O comando document.getElementById('teste') é escrito em linguagem JavaScript.

Usa-se a tag <code> no texto código, uma fonte mono-espaça (com espaços iguais entre cada letra) deixa claro o significado daquele texto como um código.

num = int(input('Digite um número')) if num % 2 == 0: print(f'O número {num} é PAR') else: print(f'O número {num} é ÍMPAR') print('Fim do programa')

O código acima ficou sem os espaçamentos (indentação) igual ao digitado, para resolver isso é necessário colocar a tag <code> dentro da tag <pre>. Aí o código ficará dessa forma:

    
    num = int(input('Digite um número'))
    if num % 2 == 0:
        print(f'O número {num} é PAR')
    else: 
        print(f'O número {num} é ÍMPAR')
    print('Fim do programa')

Para tirar os espaçamento (de todas as linhas ao mesmo tempo) basta selecionar o código e dar shift + tab
