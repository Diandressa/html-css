Trabalhando com fontes

Importante dizer que quando uma fonte tem duas palavras, como a Courier New, é necessário coloca-las entre aspas. Ex:

body {font-family:'Courier New';}

Veja o exemplo a seguir. É importante escolher fontes que tenham nos dispositivos. Por isso é inserido três opções no exemplo, se não houver a fonte Arial, irá ser utilizada a Helvetica, se não tiver a Helvetica será usada sans-serif (fontes sem serifas genéricas, ou seja, qualquer fonte sem serifa que o dispositivo tenha pode ser usada).

Isso se aplica a outros categorias como serif, monospace, etc. Sempre termine sua declaração de fontes seguras com um genérico, tipo sans-serif ou serif:

        
        <style>
            body {
                font-family:Arial, Helvetica, sans-serif
            }
        </style>

Isso é chamado de safe combination. É possível pesquisar quais são as safe combination. Dê uma olhada em alguns safes conbinations no link a seguir: CSS Web Safe Fonts

Tamanhos de fontes

Temos as medidas absolutas e as relativas.

  • Medidas Absolutas: cm, mm, in (polegada), px, pt (ponto), pc (paica).

As medidas absolutas não são indicadas para se trabalhar em telas. Ponto e paica são medidas da tipografia, no papel.

  • Medidas Relativas: em, ex, rem, vw, vh, %

São várias. O "em" é a medida relativa do tamanho atual da fonte (relativo ao tamanho do M maiuscúla da fonte), o "ex" é relativo à altura x de uma fonte, o rem é relativo à um tamanho fixo/raiz (r-> root) no body, temos tb o vw (viewwidth) sendo a largura em porcentagem da viewport (tamanho da minha tela), o vh é ralacionado à altura do viewport.

Peso da fonte

Usa a declação: font-weight: normal;

Existem variações de pesos: lighter, normal, bold, bolder. Obs: nem todas as fontes vão ter todas as variações de pesos.

Outra variação é a numérica, começa em 100 e vai até 900: font-weight: 100;

Variações da fonte

O font-style é para usar uma fonte em itálico: font-style: italic;

Para o sublinhado utilizar: text-decoration: underline;

Condensar declarações

Para condensar declarações usa-se shorthand para fontes, o shorthand font

Ordem do shorthand font: -font-style -font-weight -font-size -font-family

Antes:

        
        h1 {
            font-family:Arial, Helvetica, sans-serif;
            font-size: 3em;
            font-weight: bolder;
            font-style: italic;
            text-decoration: underline;
        }

Depois:

        
            h1 {
                font: italic bolder 3em Arial, Helvetica, sans-serif;
            }
