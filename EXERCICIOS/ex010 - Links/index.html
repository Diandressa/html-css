Usando links externos

Você pode acessar o meu repositório público no GitHub

Usamos o parâmetro target e rel(relativo à). No caso acima foi usado o valor _blank (em branco) pro target e o valor external (link externo) pro rel.

Você também pode acessar o meu canal no Youtube

Usando links internos

Esta é a primeira página do site. Se você quiser, pode acessar também a minha segunda página (sobre nofollow)

Nos links internos não se usa internal e sim next no parâmetro rel e prev (previous/anterior) na segunda

Visualmente não acontece nada, mas diz pros mecanismos de busca qual é a próxima página

Você também pode acessar a nossa página de notícias (sobre ./ e ../ navegação entre pastas)

Usa para nevegar em links na sua própria página a tag target com valor _self

Links para download

  • Baixar o livro em PDF
  • Livro compactado em ZIP

Pode colocar # no parâmetro href quando ainda não tiver link, ele significa link vazio

Dependendo do navegador o link só abre em outra aba. Para isso colocar o parâmetro download e como valor inseri o nome do arquivo que será feito o download.

Colocar também o parâmetro type e o valor "application/pdf"

Existe uma lista com os valore de type:
https://www.iana.org/assignments/media-types/media-types.xhtml
